# 🌌 Three.js Background System

This portfolio uses Three.js to create:

- Realtime particle rendering
- Interactive mouse movement
- 3D depth effect
- Animated futuristic background
- AI-inspired visual experience

The particle system is rendered using WebGL directly in the browser.
